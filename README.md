# Velocilector

**Velocilector** es un juego diseñado para mejorar la habilidad de lectura rápida y reconocimiento automático de palabras, especialmente para niños.

## Características del Juego

- **Dos Modos de Juego:**
  1. **Modo Manual:**
      - Se avanza a la siguiente palabra haciendo clic en cualquier lugar de la pantalla o tocando en dispositivos táctiles.
      - Cada clic muestra una nueva palabra.
  2. **Modo Automático:**
      - Las palabras se muestran automáticamente con un temporizador inicial de 5 segundos.
      - El tiempo entre palabras disminuye un 5% cada 20 palabras para aumentar el desafío.

- **Interfaz Intuitiva:**
  - La palabra aparece grande y centrada en pantalla para facilitar la lectura.
  - Un contador en la parte superior muestra el número de palabras leídas.
  - Una barra de progreso indica el tiempo restante en el Modo Automático.

- **Logo Interactivo:**
  - Al hacer clic en el logo del juego (arriba a la izquierda), se reinicia el juego y se vuelve al menú principal.

## Tecnologías Utilizadas

1. **HTML:** Estructura base de la aplicación.
2. **CSS:** Estilos visuales, diseño responsivo y esquema de colores.
3. **JavaScript:**
   - Manejo de eventos para clics y teclas.
   - Temporizadores y lógica de juego.
   - Generación de palabras aleatorias.

## Cómo Usar

1. **Inicio:**
   - En el menú principal, selecciona un modo de juego haciendo clic en "Modo 1: Manual" o "Modo 2: Automático".

2. **Modo Manual:**
   - Haz clic o toca la pantalla para cambiar a la siguiente palabra.

3. **Modo Automático:**
   - Las palabras cambian automáticamente con un temporizador.
   - Observa la barra de progreso para saber cuánto tiempo queda hasta la siguiente palabra.

4. **Reinicio:**
   - Haz clic en el logo en la esquina superior izquierda para reiniciar el juego y volver al menú principal.

## Objetivo

**Velocilector** busca fomentar la práctica y el aprendizaje de lectura rápida de una manera divertida, interactiva y accesible, ideal para niños y principiantes en la lectura.
